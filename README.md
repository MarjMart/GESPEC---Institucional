# GESPEC - Sistema de Gestão para APMs
Portfólio / TCC)

## Sobre o Projeto

O GESPEC (Gestão de Prestação de Contas) é uma aplicação web desenvolvida para auxiliar na gestão administrativa e financeira das Associações de Pais e Mestres (APMs).

O sistema foi criado com o objetivo de centralizar informações financeiras, facilitar o controle de receitas e despesas, acompanhar eventos, gerar relatórios gerenciais e proporcionar maior transparência na administração dos recursos da associação.

O projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso Técnico em Desenvolvimento de Sistemas da ETEC Dra. Ruth Cardoso.

---

## Objetivos

- Organizar o controle financeiro das APMs.
- Centralizar informações administrativas e financeiras.
- Facilitar a prestação de contas.
- Melhorar a transparência na gestão dos recursos.
- Apoiar a tomada de decisões por meio de relatórios gerenciais.

---

## Principais Funcionalidades

### Gestão Financeira
- Cadastro de receitas e despesas.
- Controle de movimentações financeiras.
- Consulta de saldos.

### Contas Bancárias
- Gerenciamento de contas.
- Acompanhamento de saldos disponíveis.

### Gestão de Eventos
- Controle financeiro de eventos.
- Registro de entradas e saídas vinculadas aos eventos.
- Apuração de resultados.

### Relatórios
- Emissão de relatórios financeiros.
- Consulta de movimentações.
- Apoio à prestação de contas.

### Auditoria
- Registro de histórico das operações.
- Rastreabilidade das movimentações realizadas no sistema.
